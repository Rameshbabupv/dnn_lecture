# ============================================
# Troubleshooting
# ============================================
#
# Issue: "No module named 'ultralytics'"
# Solution: pip install --upgrade ultralytics
#
# Issue: "Torch not compiled with CUDA"
# Solution: In Colab, Runtime → Change runtime type → GPU
#
# Issue: "Out of memory during training"
# Solution: Reduce batch size in training notebook (16 → 8 → 4)
#
# Issue: "pycocotools installation fails"
# Solution: On Windows, install Visual C++ Build Tools first
#          On Linux/Mac, usually works without issues
#          Or skip: pip install --no-deps pycocotools
#
# Issue: "Model download fails"
# Solution: Check internet connection, or manually download from:
#          https://github.com/ultralytics/assets/releases
#
# Issue: "Training is very slow"
# Solution: Enable GPU in Colab, or reduce dataset size for testing
